CAPSUL installation¶

Installing with BrainVISA¶

CAPSUL is the new pipelining system of the BrainVISA suite. It can therefore be installed with BrainVISA installer.

Installing without BrainVISA¶

The latest stable Capsul version can be installed with pip. Please refer to the pip documentation to get full installation possibilities. Here are some examples for most frequent situation.

Install in a directory¶

To install Capsul in a self-content directory, we recommend to use virtualenv.

>>> virtualenv my_python_modules
>>> my_python_modules/bin/pip install --upgrade capsul

Install on Linux for a single user (without administrator privilege)¶

>>> pip install --user --upgrade capsul
