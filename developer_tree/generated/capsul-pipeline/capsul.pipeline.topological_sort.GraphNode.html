Simple Graph Node Structure

Attributes

name (str) the node name
meta (object) a python object stored in the node
links_to (list) object to store the graph edges: sucessor
links_from (list) object to store the graph edges: predecessor
links_to_degree (int) degree of the node regarding the successors
links_from_degree (int) degree of the node regarding the predecessors
